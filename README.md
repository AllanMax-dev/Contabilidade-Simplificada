# Ana Beatriz Contabilidade Simplificada

Site institucional estático desenvolvido com HTML, CSS e JavaScript.

## Estrutura

```text
.
├── assets/
│   ├── css/
│   │   └── styles.css
│   ├── images/
│   │   ├── Ana.jpeg
│   │   ├── Ana2.jpeg
│   │   └── logo.png
│   └── js/
│       └── main.js
└── index.html
```

## Arquivos principais

- `index.html`: conteúdo, semântica e metadados da página.
- `assets/css/styles.css`: identidade visual, componentes e responsividade.
- `assets/js/main.js`: menu móvel, links do WhatsApp, animações e ano do rodapé.

## Configuração

O número do WhatsApp está definido na constante `WHATSAPP_NUMBER`, em `assets/js/main.js`.

Para visualizar localmente, abra `index.html` no navegador ou use um servidor HTTP estático.
