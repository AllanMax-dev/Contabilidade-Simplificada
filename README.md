# Ana Beatriz Contabilidade Simplificada

Landing page institucional desenvolvida para a marca **Ana Beatriz Contabilidade Simplificada**, com foco em apresentação profissional, geração de autoridade e conversão de visitantes em contatos pelo WhatsApp.

O projeto foi construído como uma página estática, sem dependências externas de build, utilizando **HTML**, **CSS** e **JavaScript puro**.

## Visão geral

A página apresenta os serviços contábeis e tributários da profissional Ana Beatriz, destacando especialização em:

- Reforma tributária;
- Imposto de renda;
- Recuperação de créditos fiscais;
- Consultoria tributária;
- Planejamento tributário;
- Atendimento contábil com linguagem clara e acessível.

O objetivo principal da landing page é transmitir confiança, explicar os serviços de forma simples e direcionar o visitante para uma conversa no WhatsApp.

## Funcionalidades

- Layout responsivo para desktop, tablet e celular;
- Menu fixo com efeito ao rolar a página;
- Menu mobile com abertura e fechamento por botão;
- Botões de contato integrados ao WhatsApp;
- Mensagens personalizadas para cada chamada de ação;
- Animações suaves de entrada durante a rolagem;
- Estrutura semântica em HTML;
- Metadados para SEO básico;
- Open Graph para compartilhamento em redes sociais;
- Dados estruturados com Schema.org para serviço contábil;
- Ano automático no rodapé;
- Identidade visual com paleta em azul-marinho, creme e dourado.

## Tecnologias utilizadas

- HTML5;
- CSS3;
- JavaScript;
- Google Fonts;
- Schema.org JSON-LD.
